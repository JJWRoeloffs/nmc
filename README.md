# NMC

A personal repository used to toy with neural networks for model checking, building on top of the work of Giacobbe et al. [1],[2]

[1] Giacobbe, M., Kroening, D., Pal, A., & Tautschnig, M. (2024). Neural model checking. Advances in Neural Information Processing Systems, 37, 86375-86398.
[2] Giacobbe, M., Kroening, D., Pal, A., & Tautschnig, M. (2025, September). Let a neural network be your invariant. In The Thirty-ninth Annual Conference on Neural Information Processing Systems.
